## 2. Описание предметной области
Система "Каталог товаров" — это упрощенный аналог витрины интернет-магазина. Приложение должно выводить список товаров (с фотками, ценами и названиями), давать возможность фильтровать их по категориям и добавлять в виртуальную корзину. Серверной части пока нет, все данные будут обрабатываться на стороне клиента.

## 3. Пользователи
- **Обычный юзер (покупатель):** листает каталог, ищет товары, кликает "в корзину".
- **Админ (в перспективе):** добавляет новые карточки товаров в массив данных.

## 4. Главные сущности
- **Товар:** id, название, цена, фотка, категория.
- **Категория:** просто группировка (например, одежда, электроника).
- **Корзина:** массив того, что накликал юзер, плюс счетчик итоговой суммы.

## 5. Сценарии использования (Use Cases)
1. Открытие страницы: загружается интерфейс и сетка товаров.
2. Фильтрация: юзер жмет галочку "Обувь", остаются только нужные карточки.
3. Покупка: клик по кнопке на карточке -> счетчик корзины в шапке увеличивается.

## 6. История реализации (ЛР №1)
Создана базовая структура. Сверстаны шапка, боковое меню с фильтрами, сетка для товаров и подвал. Файлы разбиты логически, заведен гит-репозиторий.
